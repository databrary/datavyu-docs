===============
Rename a Column
===============

Datavyu provides four different ways to rename a column using the
Datvyu user interface.

Rename Directly from the Spreadsheet
------------------------------------

#. Open the Datavyu spreadsheet.

#. Double click on the column name. The :guilabel:`New variable name`
   window will open.

#. Input the new name for the column, and press Enter or click on
   :guilabel:`OK`.

Rename Using the Spreadsheet Menu
---------------------------------

You can also change a column's name from the :guilabel:`Spreadsheet`
menu.

#. Open the Datavyu spreadsheet.

#. Select the variable whose name you wish to change. Selected
   variables have a blue background.

#. Select the :guilabel:`Spreadsheet` menu, and then :guilabel:`Change
   Variable Name`.The :guilabel:`New variable name`
   window will open.

#. Input the new name for the column, and press Enter or click on
   :guilabel:`OK`.

Rename From the Column List
---------------------------

Using the :guilabel:`Column List` is another way you can change a
column's name. The :guilabel:`Column List` also has the advantage of
enabling you to change multiple column names in one place.

#. Open the Datavyu spreadsheet.

#. Open the :guilabel:`Spreadsheet` menu, and then select
   :guilabel:`Column List`.

#. Double click on the name of the column whose name you want to
   change, type in a new column name, and press Enter.

Rename Matrix Columns Using Vocab Editor
----------------------------------------

Datavyu's :guilabel:`Vocab Editor` is primarily used for configuring
:term:`matrix columns' <matrix column>` :term:`codes <code>`, but you
can also edit your columns' names. You cannot use the :guilabel:`Vocab
Editor` to edit :term:`text column` or :term:`nominal column` names
because they are not listed in the :guilabel:`Vocab Editor`.

#. Open the Datavyu spreadsheet.

#. Open the :guilabel:`Spreadsheet` menu, and then select
   :guilabel:`Vocab Editor`.

#. Double click on the name of the column whose name you want to
   change, and type a new column name. The spreadsheet will update
   automatically.
