We have produced several videos that help illustrate Datavyu's strengths
and guide users through common tasks:

Datavyu Components & Playback
~~~~~~~~~~~~~~~~~~~~

.. raw:: html

   <video controls width="100%" height="auto">
   <source src="https://nyu.databrary.org/slot/45710/0,704610/asset/238623/download?inline=true" type="video/mp4">

Learn about Datavyu's media player and video controller. Discover the power of Datavyu's fingertip playback control. For more information see :doc:`/guide/getting-started` and :doc:`/guide/controller`.

Time-Lock Events Codes to Video: Cells & Coding Spreadsheet
~~~~~~~~~~~~~~~~~~~~

.. raw:: html

   <video controls width="100%" height="auto">
   <source src="https://nyu.databrary.org/slot/45710/705566,1639961/asset/238625/download?inline=true" type="video/mp4">
   </video>

Learn how Datavyu time-locks video events to the coding spreadsheet. Dive into how a coding column and a "cell" within a coding column work. See examples of time-locking events to video. For more information see :doc:`/guide/spreadsheet` and :doc:`/guide/tutorials/configure-datavyu-codes`.

Time-Lock Events Codes to Video: Coding Timestamps
~~~~~~~~~~~~~~~~~~~~

.. raw:: html

   <video controls width="100%" height="auto">
   <source src="https://nyu.databrary.org/slot/45710/1640961,2576493/asset/238627/download?inline=true" type="video/mp4">

Learn how to use Datavyu’s fingertip control functions to time-lock video to the coding spreadsheet. Explore three ways of inserting timestamps with appropriate use cases. For more information see :doc:'/guide/tutorials/add-cells'.
