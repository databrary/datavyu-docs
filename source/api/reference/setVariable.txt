Definition
----------

.. method:: setVariable(*args)

   Replaces a column in the database with a new column of the
   same name as the method's arguments. If no column with the same
   name exists in the database, it will instead create a new column.

   .. table-start

   .. list-table:: 
      :widths: 20, 40, 50
      :header-rows: 1
   
      * - Parameter
        - Type
        - Description
      * - ``name`` (**optional**)
        - String
        - Name of the column that is being created
      * - ``var`` (**required**)
        - Ruby variable
        - Ruby container of the column that is being put into the
          database (modified output of :meth:`createNewVariable` or
          :meth:`getVariable`)

   .. important:: 

      You must specify the ``var`` argument to :meth:`setVariable()`.
      If you are also passing the ``name`` argument, the order of arguments
      must be ``name`` followed by ``var``.

   .. table-end

   .. admonition:: Returns

      None

   .. admonition:: Example

      The following example retrieves the Datavyu column "trial" and 
      assigns it to a Ruby variable called ``trial``.

      .. code-block:: ruby

         require 'Datavyu_API.rb'
         begin
            trial = getVariable("trial")
            <some modifications to trial>
            setVariable("trial", trial)
         end
