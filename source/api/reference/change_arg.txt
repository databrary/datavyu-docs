============
change_arg()
============

Definition
----------

``change_arg(arg, val)``

Changes the value of an argument in a cell.

.. table-start

.. list-table:: 
   :widths: 20, 40, 50
   :header-rows: 1
   
   * - Argument
     - Type
     - Description
   * - ``arg``
     - String **or** Ruby variable from :function:`getVariable`
     - Name of the argument that you are updating.
   * - ``val``
     - String, Integer, etc.
     - Value to change the argument to.

.. table-end

.. admonition:: Returns

   None.

Examples
--------

The following example sets the "trial" variable's cell at position 0's
:term:`onset` to 1000ms, and then writes the change back to the database
using :function:`setVariable`.

.. code-block:: ruby

   trial = getVariable("trial")
   trial.cells(0).change_arg("onset", 1000)
   setVariable("trial", trial)
