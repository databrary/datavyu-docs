===========
setColumn()
===========

.. method:: setColumn(name, var)

   Replaces a column in the spreadsheet with a new column of the
   same name as the method's argumnets. If no column with the same
   name exists in the spreadsheet, it will instead create a new column.

   .. table-start

   .. list-table:: 
      :widths: 20, 40, 50
      :header-rows: 1
   
      * - Parameter
        - Type
        - Description
      * - ``name`` (**optional**)
        - String
        - Name of the column that you are inserting
      * - ``var`` (**required**)
        - Ruby variable
        - Ruby container of the column that you are inserting into the
          spreadsheet (modified output of :meth:`createNewColumn()` or
          :meth:`getColumn()`)

   .. important:: 

      You must specify a value for the ``var`` parameter.  If you are
      also passing a value for the ``name`` parameter, the order of
      arguments **must** be ``name`` followed by ``var``.

   .. table-end

   .. admonition:: Returns

      None

   .. admonition:: Example

      The following example retrieves the Datavyu column "trial" and 
      assigns it to a Ruby variable called ``trial``.

      .. code-block:: ruby

         require 'Datavyu_API.rb'
         begin
            trial = getColumn("trial")
            <some modifications to trial>
            setColumn("trial", trial)
         end
