=============
getVariable()
=============

.. default-domain:: rb

Definition
----------

.. method:: getVariable(name)

   Retrieves a variable from the Datavyu database and assigns it to a Ruby
   object using :meth:`print_debug` 

   .. table-start

   .. list-table:: 
      :widths: 20, 40, 50
      :header-rows: 1
   
      * - Parameter
        - Type
        - Description
      * - ``name``
        - String
        - The name of the Datavyu variable you wish to retrieve

   .. table-end

   .. admonition:: Returns

      A Ruby object representation of the Datavyu variable

   .. admonition:: Example

      The following example retrieves the Datavyu variable "trial" and 
      assigns it to a Ruby variable called ``trial``.

      .. code-block:: ruby

         require 'Datavyu_API.rb'
         begin
            trial = getVariable("trial")
         end
